# Men's Watch Photos

Place your men's watch photos in this folder.

## Naming Convention:
- men-watch1.jpg
- men-watch2.jpg
- men-watch3.jpg
- etc.

## Supported Formats:
- .jpg, .jpeg, .png, .webp

## Recommended Size:
- Minimum 800x800px
- Under 2MB file size

These photos will be used for the men's products in your e-commerce store.
