# Women's Watch Photos

Place your women's watch photos in this folder.

## Naming Convention:
- women-watch1.jpg
- women-watch2.jpg
- women-watch3.jpg
- etc.

## Supported Formats:
- .jpg, .jpeg, .png, .webp

## Recommended Size:
- Minimum 800x800px
- Under 2MB file size

These photos will be used for the women's products in your e-commerce store.
