# Watch Photos Directory

This folder contains all your watch product images organized by category.

## Folder Structure:
```
watches/
├── men/      - Men's watch photos
├── women/    - Women's watch photos
└── unisex/   - Unisex watch photos
```

## How to Add Your Photos:

1. **Men's Watches**: Place in `/men/` folder
   - Name: men-watch1.jpg, men-watch2.jpg, etc.

2. **Women's Watches**: Place in `/women/` folder  
   - Name: women-watch1.jpg, women-watch2.jpg, etc.

3. **Unisex Watches**: Place in `/unisex/` folder
   - Name: unisex-watch1.jpg, unisex-watch2.jpg, etc.

## Image Requirements:
- **Format**: .jpg, .jpeg, .png, .webp
- **Size**: Minimum 800x800px recommended
- **File Size**: Under 2MB for faster loading
- **Quality**: Clear, well-lit photos on neutral backgrounds

## After Adding Photos:
Your watch photos will automatically appear in:
- Shop page with filtering
- Product detail pages
- Homepage carousels
- Cart and checkout

The system is already configured to use these images once you add them!
