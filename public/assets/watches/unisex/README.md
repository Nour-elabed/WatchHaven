# Unisex Watch Photos

Place your unisex watch photos in this folder.

## Naming Convention:
- unisex-watch1.jpg
- unisex-watch2.jpg
- unisex-watch3.jpg
- etc.

## Supported Formats:
- .jpg, .jpeg, .png, .webp

## Recommended Size:
- Minimum 800x800px
- Under 2MB file size

These photos will be used for the unisex products in your e-commerce store.
